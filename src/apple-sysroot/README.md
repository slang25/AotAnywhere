# Apple linker stub sysroot

Minimal Mach-O linker stubs (`.tbd` files) for the Apple system libraries and
frameworks that .NET Native AOT binaries link against. They let `zig cc`
cross-compile for macOS from Linux and Windows hosts, where no Apple SDK is
available. Stubs only matter at link time; at run time the real libraries on
the target Mac are loaded.

These files are **not** copies of Apple SDK stubs. Each one is generated by
`eng/generate-apple-sysroot.cs`, which collects the undefined symbols that the
MIT-licensed .NET runtime packs actually reference and writes only those
symbol names into freshly authored tbd-v4 documents (the same approach zig
itself uses for its bundled `libSystem` stub, and mingw-w64 uses for Windows
import libraries).

To regenerate (requires macOS with Xcode and zig on PATH):

```sh
dotnet run eng/generate-apple-sysroot.cs
```

Update `PackVersions` in that script when new .NET versions ship.
